Instalado o Angular

Para se usar o Angular iniciamente é necessário fazer as instalações iniciais. Para fazer a instalação em seu computador rode o seguinte comando no seu prompt:

npm install g @angular/cli

O Angular CLI irá te permitir criar projetos, modulos, componentes e demais afins de formas mais rápida e prática.


Após a instalação do Angular CLI, você pode criar programas utilizando Angular. Inicialmente entre no local da memória em que deseja criar um novo projeto, e utilize o seguinte comando:

ng g new nomeDoProjeto

Este comando instalará todos os pacotes e arquivos necessários para o funcionamento do Angular automaticamente, além de criar os primeiros arquivos necessários para iniciar um projeto.

Vale lembrar, que para usar o Angular você precisa inicialmente ter instalado o Node JS em seu computador.


Após a criação do projeto, você poderá executar o seu projeto utilizando o comando dentro da pasta que seu projeto foi criado:

ng serve

Por padrão o Angular iniciara utilizando a porta local 4200 do seu computador, você poderá acessar a página através do seu navegador e entrando no link: localhost:4200
