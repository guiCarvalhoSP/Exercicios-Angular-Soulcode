Os Serviços

Os serviços (services) são onde ficará toda a parte lógica das suas páginas. As classes e funções dos servços são chamadas pelos componentes, executam alguma função e depois retornam o resultado para o componente. Você pode criar a quantidade de services que achar necessário para seu projeto, porém o ideal é um service para cada grupo de funcionalidades relacionadas.

Você pode criar um service de forma rapida usando o seguinte código:

ng g service nomeDoServico

Ou use a versão simplificada:

ng g s nomeDoServico

Você também pode especificar em qual pasta quer que seja criada os seus serviços, colocando o nome da pasta que deseja que seja criado o serviço, seguido por uma barra, como no exemplo a seguir:

ng g c services/nomeDoServico
