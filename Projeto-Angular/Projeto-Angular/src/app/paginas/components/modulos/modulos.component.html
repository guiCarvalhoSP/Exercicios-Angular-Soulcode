Os Módulos

Os módulos em Angular são uma forma de agrupar componentes, diretivas, pipes e serviços que possuem algum relacionamento. De uma forma mais simples de entender, um módulo é como se fosse um grupo de funções que trabalham juntos para rodar alguma determinada aplicação. São nos módulos que você exporta as funcionalidades para outras pastas e arquivos, além de importar as funções que irá utilizar em sua aplicação.

Ao criar um projeto Angular, o projeto irá vir com um Módulo principal, onde quaisquer outros módulos devem ser importados no local. Caso deseje criar um novo módulo, utilize o código:

ng g module nomeDoModulo

Ou use a versão simplificada:

ng g m nomeDoModulo

Você também pode especificar em qual pasta quer que seja criada os seus módulos, colocando o nome da pasta que deseja que seja criado o módulo, seguido por uma barra, como no exemplo a seguir:

ng g c pastaEscolhida/nomeDoModulo

Sempre que criar um novo módulo se lembre de declara-lo no módulo principal (app.module.ts)
