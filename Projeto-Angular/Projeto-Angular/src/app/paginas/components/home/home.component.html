O Angular

O angular é um framework de código fonte aberto feito em lingaugem TypeScript desenvolvido pela Google. O Angular atual é uma reescrição do AngularJs.

O objetivo do Angular é deixar uma pagina mais dinâmica, criando uma single page, onde quando o usúario faz alguma requisição, ao ínves de recarregar toda a página, ele irá carregar apenas aquilo que é necessário ser mudado. O Angular torna o trafégo de dados menor devido a carregar apenas o que for necessário.

Ele oferece uma grande variedade de estruturas para fazer o desenvolvimento do front-end.
