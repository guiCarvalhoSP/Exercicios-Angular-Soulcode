Diretivas

As diretivas são marcadores de elementos DOM, elas são usadas para que o Angular anexe um determinado comportamento para aquele elemento DOM.


Os principais tipos de diretivassão o ngStyle, ngClass, ngIf, ngSwitch e ngFor. Para usá-las, basta colocar dentro das declarações do elemento a sua sintaxe e os demais dados que cada uma necessita. Veja a seguir como usar elas:

ngStyle

O ngStyle serve para alterar em tempo real os valores css de um elemento DOM. sua sintaxe é [ngStyle]= "{{'{'}} {{'}'}}" .


ngClass

Está diretiva tem como função atribuir ou retirar uma classe de algum elemento DOM. [ngClass]= "{{'{'}} {{'}'}}" .


ngIf

Tem funcionalidade semelhante ao If de outras linguagens. Sua função é adicionar ou não um comportamento ao elemento DOM caso uma expressão booleana seja verdadeira ou falsa. *ngIf= "expressaoBooleana" .


ngSwitch

Comportamento semelhante ao ngIf. Sua função é adicionar ou não um comportamento ao elemento DOM caso uma expressão booleana seja verdadeira ou falsa, sendo a sua principal diferença do ngIf o fato de ter um ngSwitchCase e um NgSwitchDefault. o NgSwitch recebe o valor que será testando, os ngSwitchCase farão uma verificação da expressão para saber se ela é verdadeira ou não (caso seja, executará a modificação no DOM), enquanto o NgSwitchDefault fará alguma função caso nenhum dos Cases retorne verdadeiro. A sintaxe de cada uma das 3 é: [ngSwitch]= 'atributo' , *ngSwitchCase= "expressaoBooleana", *ngSwitchCase= "valorAtribuido" .


ngFor

O ngFor tem como função criar um loop para arrays que serão mostrados na tela. Sua sintaxe é: *ngFor= "let value of values" .
