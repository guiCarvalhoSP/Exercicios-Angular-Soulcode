Os componentes

Os componentes são partes importantes do Angular, eles são as partes/páginas que vão ser mostradas ao usúario.

Os componentes consistem em um html, css, um component.ts e um component.ts para testes. O arquivo .ts serve para executar as funções da página, que normalmente ficam localizas no service.

Você pode criar um componente novo para cada "página" que você deseja criar. Para criar uma pasta onde terá os seus componentes digite o seguinte código:

ng g component nomeDoComponente

Ou use a versão simplificada:

ng g c nomeDoComponente

Você também pode especificar em qual pasta quer que seja criada os seus componentes, colocando o nome da pasta que deseja que seja criado o componente, seguido por uma barra, como no exemplo a seguir:

ng g c pastaEscolhida/nomeDoComponente

Após criar um componente, especifique no modulo os componentes que irá utilizar. Para isso coloque em declarations o nome da classe do componte (presente no arquivo ts criado), e faça sua importação no topo da página.
